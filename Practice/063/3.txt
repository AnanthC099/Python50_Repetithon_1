### 2) Quotes inside quotes + escaping (backslash `\`)

**Concepts covered**

* `'...'` can contain `"` without escaping
* `"..."` can contain `'` without escaping
* To include the *same* quote type inside, use **backslash escape**: `\'` or `\"`

```text
>>> # Singly quoted string can contain double quote as a data
>>> s = 'This is "supposedly" a time for study'
>>> print(s)
This is "supposedly" a time for study
>>> # Doubly quoted string can contain single quote as a data
>>> s = "This is John's room"
>>> print(s)
This is John's room
>>> # You will require backlash (escape) in order to put " inside doubly quoted string
>>> # You will require backslash (escape) in order to put ' inside singly quoted string
>>> s = 'This is John\'s room'
>>> print(s)
This is John's room
>>> s = "This is \"Supposedly\" a time for study"
>>> print(s)
This is "Supposedly" a time for study
```

**Concepts covered**

* `print(..., ..., ...)` prints values separated by spaces
* Old-style formatting: `%d`, `%f`, etc.
* Normal string `'My team is {L}'` does **not** substitute values
* f-string `f'...'` substitutes expressions inside `{}`

## C) Splitting: `split()` and `rsplit()`

**Concepts covered**

* `split(':')` splits from left
* `rsplit(':')` splits from right
* `maxsplit=2` means “split at most 2 times”

  * `split(..., maxsplit=2)` keeps the *rest* in the last element
  * `rsplit(..., maxsplit=2)` keeps the *rest* in the first element

```text

# split() and rsplit()
        
s = 'abc:pqr:xyz:lmn'
        
L = s.split(':')
        
L

s

s = 'abc:pqr:xyz:lmn:uvw:rst:vbn:mno'

s.rsplit(':')

s.split(':')
        
['abc', 'pqr', 'xyz', 'lmn', 'uvw', 'rst', 'vbn', 'mno']
s.split(':', maxsplit=2)
```
